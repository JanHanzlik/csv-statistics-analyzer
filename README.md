# 📊 CSV Statistics Analyzer

Simple Python tool that analyzes CSV datasets and generates useful statistical summaries.

The program reads CSV files, detects numeric columns, calculates statistics and exports the results to an Excel report.

---

## 📊 Features

* Reads CSV files
* Automatically detects numeric columns
* Calculates statistics:

  * count
  * missing values
  * sum
  * mean
  * median
  * min
  * max
  * standard deviation
* Exports raw data and statistics to an Excel report
* Can generate sample CSV test data

## ⚙ Installation

Clone the repository and install dependencies:

```bash
pip install -r requirements.txt
```

---

## ▶ Usage

Generate sample test data:

```bash
python generate_test_data.py
```

Run the CSV analyzer:

```bash
python analyze_csv.py test_data.csv
```

Run with custom output file:

```bash
python analyze_csv.py test_data.csv custom_report.xlsx
```

---

## 📈 Output

The program generates:

* `output_report.xlsx` — Excel report containing:

  * **Raw Data** sheet
  * **Statistics** sheet
